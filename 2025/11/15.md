## わかったこと
- `TextChoices` は「DBに保存する値」と「表示用ラベル」をまとめて扱えるため従来のタプルより安全で見通しが良い  
- Enum を使うことで `Priority.HIGH` など補完が効きタイプミスの心配がなくなる  
-  `date` フィルタは `F` で英語月名になるが、`m` や `n` を使うと日本語の「○月」にできる  
- `n`（ゼロなし月）、`j`（ゼロなし日）などフォーマット記号は PHP 由来のルールでシンプル  
- Create / Update / Delete 用の Class-based View に `template_name`・`fields`・`success_url` を指定する流れがつかめた  
- `request.method == "POST"` を正しく使い分ければフォーム送信処理を簡潔に書ける
